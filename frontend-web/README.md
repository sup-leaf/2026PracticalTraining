# 校园集市 - Web管理端

## 技术栈
- Vue 3
- Vite
- Element Plus
- Vue Router

## 项目结构
```
frontend-web/
├── src/
│   ├── api/         # API 接口
│   ├── router/      # 路由配置
│   ├── views/       # 页面组件
│   ├── App.vue      # 根组件
│   └── main.js      # 入口文件
├── index.html
├── vite.config.js
└── package.json
```

## 安装启动

```bash
# 安装依赖
cd frontend-web
npm install

# 启动开发服务器
npm run dev
```

访问 http://localhost:3000

## 功能页面
- 登录页
- 首页（侧边栏导航）
- 岗位列表（搜索、投递）
- 发布岗位
- 投递管理
