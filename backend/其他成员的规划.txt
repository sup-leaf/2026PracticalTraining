# 校园集市项目 后端任务拆解（Java+SpringBoot 版）
适配你立项书：**小程序学生端 + Vue3 企业/教师Web管理端**，按**模块→细分任务→你要做的事**拆好，直接可以分给自己做进度排期。

## 一、基础工程搭建（先做，1天搞定）
1. 初始化 SpringBoot 项目，配置依赖
   - SpringBoot 基础、Web、MyBatis-Plus、MySQL、Redis
   - 接口文档 Knife4j/Swagger
   - 工具类：日期、加密、统一返回结果、全局异常处理
2. 项目分层架构搭建
   - entity / dto / vo / controller / service / mapper / config / util
3. 全局配置
   - 跨域配置、分页配置、Jackson 时间格式化
   - 统一响应封装（Result 统一返回体）
   - 全局异常捕获、错误码定义
4.  Redis 整合
   - 连接配置、工具类封装（存验证码、登录token、缓存热门数据）

## 二、权限与登录认证模块（核心基础）
### 功能任务
1. 多角色设计：**学生 / 教师 / 企业HR / 管理员**
2. 登录接口
   - 学生：学号+密码 / 学号验证码登录
   - 教师/企业/管理员：账号密码登录
3. JWT + 拦截器 登录鉴权
   - 生成token、校验token、过期刷新
   - 接口权限拦截：不同角色只能访问对应接口
4. 实名认证相关
   - 学号、身份绑定接口
   - 敏感数据**脱敏返回**（手机号、绩点部分隐藏）

## 三、企业入驻 & 实习招聘模块（立项核心模块）
### 1. 企业管理
- 企业入驻申请提交接口
- 管理员审核企业：通过/驳回接口
- 企业信息CRUD、资质资料存储
### 2. 岗位管理
- 企业发布/编辑/下架实习岗位
- 岗位列表分页、条件筛选（专业、薪资、行业、时间）
- 岗位详情查询
### 3. 简历 & 投递流程
- 学生电子简历：新增/编辑/查询
- 岗位投递接口（一人一岗防重复投递）
- 企业查看投递简历列表
- 投递状态流转：**待查看→面试中→已录用→已拒绝** 状态更新接口
### 4. 智能初筛（简易版）
- 按条件：专业、GPA、四六级、技能标签 做后台筛选接口

## 四、教师科研项目选人模块
1. 教师发布科研项目/科研助理招募
2. 项目列表、条件筛选、详情查询
3. 学生申请加入科研项目
4. 教师审核通过/驳回申请
5. 学生科研参与记录存档接口

## 五、科创竞赛组队协作模块
1. 发布竞赛组队招募（队长招人）
2. 组队列表、标签筛选（技能、专业、赛事类型）
3. 学生申请入队、队长审核
4. 组队状态管理：招募中/已满员/已结束
5. 技能标签匹配后台基础接口（关键词匹配）

## 六、实习全过程管理 & 官方证明模块
1. 学生实习入职报备、实习日志提交
2. 实习在岗记录、周期统计
3. 企业/导师实习评分接口
4. 后台生成电子实习证明数据接口
5. 管理员查看全院实习统计、专业实习率

## 七、文件上传 & 云存储模块
1. 对接**腾讯云COS**
2. 封装通用文件上传接口
   - 简历附件、实习证明照片、竞赛作品、企业资质图片
3. 文件访问权限控制、删除接口

## 八、数据大屏 & 统计看板接口
给管理端Web大屏提供所有统计接口：
- 入驻企业数、岗位总数
- 学生实习达成率、各专业分布
- 科研项目数量、竞赛组队参与人数
- 岗位投递热度、预警数据接口

## 九、公共辅助功能
1. 短信验证码接口（登录、找回密码）
2. 验证码Redis缓存、有效期控制
3. 系统公告、通知消息接口
4. 定时任务（SpringTask）
   - 过期岗位自动下架
   - 实习周期到期提醒
   - 每日数据统计归档

## 十、联调 & 测试任务
1. 给前端提供所有接口文档
2. 配合小程序、Vue管理端联调Bug
3. 接口压力简单测试、参数校验完善
4. 敏感数据加密、隐私保护处理
